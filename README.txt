================================================================
Descrição geral do Projeto
================================================================

O Sistema de Web Services Consumer, foi desenvolvido usando as seguintes tecnologias:

Interface:Primefaces versão 3.1.1 e JSF (Java Server Faces versão 2.1)
Tema: hot-sneaks-1.0.3

Servidor Web: Apache Tomcat 7.0.1.4.0

Sistema Operacional: Ubuntu Linux versão 10.10 kernel 2.6.35-30-generic-pae

IDE: NetBeans versão 7.0.1

Arquitetura: MVC (Model, View, Controller)

Todos os campos são de preenchimento obrigatório
